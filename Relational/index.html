Relational operations module

This module allows the user to perform basic relational operations. It provides functions to compare two numbers using standard relational operators.

Examples:

>>> from relational_operations import relations
>>> relations.equal(4, 4)
True
>>> relations.greater_than(5, 3)
True
>>> relations.less_than(3, 5)
True
The module contains the following functions
  • equal(a, b) - Returns True if both numbers are equal.
  • not_equal(a, b) - Returns True if both numbers are not equal.
  • greater_than(a, b) - Returns True if the first number is greater than the second.
  • less_than(a, b) - Returns True if the first number is less than the second.
  • greater_than_or_equal(a, b) - Returns True if the first number is greater than or equal to the second.
  • less_than_or_equal(a, b) - Returns True if the first number is less than or equal to the second.

def equal(a: Union[float, int], b: Union[float, int]) -> bool:
    """Check if two numbers are equal.

    Examples:
        >>> equal(4, 4)
        True
        >>> equal(4.0, 5.0)
        False

    Args:
        a: The first number to compare.
        b: The second number to compare.

    Returns:
        True if `a` is equal to `b`, False otherwise.
    """
